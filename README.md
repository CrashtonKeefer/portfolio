# Ashton's Portfolio

A modern, animated portfolio website showcasing Discord bot development, UI/UX design, and Roblox integrations.

## Features

✨ **Interactive Animations**
- Smooth scroll effects
- Parallax scrolling
- Floating animations
- Particle effects on click
- 3D hover effects on project cards
- Typing animation effect

🎨 **Dark Theme with Blue Accents**
- Modern dark gradient background
- Cyan and blue color scheme
- Glassmorphism effects
- Responsive design

📱 **Fully Responsive**
- Mobile-first design
- Hamburger menu for mobile
- Optimized for all screen sizes

🚀 **Performance**
- Lightweight and fast-loading
- Smooth animations with GPU acceleration
- Intersection Observer for efficient animations

## Tech Stack

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- GitHub Pages hosting

## Sections

1. **Hero** - Eye-catching introduction with animated floating cube
2. **Projects** - Showcase of featured work with hover effects
3. **Skills** - Organized skills and technologies
4. **Contact** - Easy ways to get in touch

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0099ff;
    --secondary-color: #00d4ff;
    /* ... more colors ... */
}
```

### Content
Edit the HTML in `index.html` to update:
- Your name and intro
- Project details
- Skills list
- Contact information

### Animations
Modify the JavaScript in `script.js` to customize:
- Animation speeds
- Mouse effects
- Scroll effects
- Particle behavior
